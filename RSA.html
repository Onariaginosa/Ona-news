RSA Project

Written Response

We created a RSA encrypter and decrypter using Tkinter in python. The purpose of the program is to be able to encrypt and decrypt messages with any public or private key and be able to save or reset those values. In the video I first demonstrate how to use the program. You insert the public or private key into the correct boxes and put the message you want to encrypt or decrypt into the correct box. Then you press either the decryption or encryption button. You can also save or open a file. You would go into the file tab and click on either open or save. When you save you values, there is a message box that tells you if you have saved it correctly. There is also an exit and clear values tab inside of the other cascade. Those tabs exit from the program and clear your values, respectively.
